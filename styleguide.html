Interactive Elements

In-text link

CSS selector:

text-decoration: none;
color: rgb(66,104,124);

Sample code:

<a href="#" class="hyperlink">in-text link</a>

Rendered element:

in-text link

Radio Button with Label

CSS selector:

Sample code:

<input type="radio" name="color">Red

Rendered element:

Red

Image as a Link

CSS selector:

Sample code:

<a href="#"><img src="images/sfu.jpg" alt="sfu icon"></a>

h4 {margin-bottom: 0.25rem; font-size: 1.45rem;}

<h2>Heading 1</h2>
<h3>Heading 2</h3>
<h4>Heading 3</h4>
<h5>Heading 4</h5>
